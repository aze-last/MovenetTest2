# CellWatch AI

CellWatch AI is an advanced institutional monitoring system designed for real-time behavior analysis and contraband detection. Utilizing MoveNet for pose estimation and YOLOv8 for object detection, the system provides automated surveillance to enhance security in correctional facilities and other high-security environments.

## Features
- **Real-time Monitoring**: Multi-camera feed with automated motion gating.
- **Behavior Analysis**: Detects aggressive movements, falls, and suspicious behavior.
- **Contraband Detection**: Identifies prohibited items like weapons and mobile devices.
- **Incident Recording**: Automated video capturing and evidence logging.
- **Intuitive Dashboard**: System health tracking and real-time statistics.
- **Modern UI**: Dark-themed management interface built with CustomTkinter.

## Technical Stack
- **AI Engines**: MoveNet (TensorFlow) & YOLOv8 (Ultralytics)
- **Framework**: Python 3.10+
- **UI Library**: CustomTkinter
- **Database**: SQLite3
- **CV**: OpenCV-python
